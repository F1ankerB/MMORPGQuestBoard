О проекте

Этот проект является учебной работой, направленной на создание интернет-ресурса для фанатского сервера одной известной MMORPG. Основные функции и особенности проекта включают:

  • Регистрация пользователей по e-mail с подтверждением через письмо.
  • Возможность создания и редактирования объявлений с картинками, видео и другим контентом.
  • Функционал откликов на объявления с уведомлениями по e-mail.
  • Приватная страница для управления откликами на свои объявления.
  • Категоризация объявлений: Танки, Хилы, ДД и другие специализации.
  • Возможность рассылки новостей пользователям.

Цель проекта - предоставить удобный и функциональный сервис для сообщества игроков MMORPG.
